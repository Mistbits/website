# Mistbits website

Static marketing site for [mistbits.com](https://www.mistbits.com), deployed through GitHub Pages.

The site is intentionally dependency-free: HTML, a shared CSS file, and optimized image assets. To preview it locally:

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`. Pull requests are checked with HTMLHint in GitHub Actions.
